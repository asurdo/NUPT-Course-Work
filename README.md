# NUPT-Course-Work

本库为内部技术交流使用
主要存放各种课程的实验以及程序设计周相关的内容

----------

* **课程实验及报告**
存放课程的实验内容（代码、报告等）
放在 Course文件夹

* **程序设计周**
相关代码、报告等
放在Week文件夹

----------

为方便交流，再此规范文件命名
Course/课程名称/题目/昵称/具体实验内容
Week/课程名称/项目名称/昵称/具体内容
